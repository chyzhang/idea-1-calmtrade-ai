# CalmTrade AI

> A calmness layer for traders: breathe now, then get chart-context coaching.

## Product Path
**Mobile app** — Traders need instant intervention during live sessions — can't switch browser tabs away from trading platform

## User Journey
User feels anxious or impulsive during a live trade → One-tap opens breathing screen with current market context → Complete 20-second guided breathing + optional chart coaching → Anxiety settles, user makes calmer trading decision instead of revenge-trading

## Screens
- **Breathe** — 20-second guided breathing exercise with haptic pacing and timer
- **Context** — AI chart coaching based on current market position and emotional state  
- **Track** — Log trades and emotional triggers to identify patterns over time

## MVP Features
**Must have:**
- 20-second guided breathing with haptic pacing and countdown
- One-tap emergency calm mode accessible from lock screen
- Basic trade logging with emotion tagging

**Next up:**
- AI chart analysis and position coaching
- Pattern recognition for emotional trading triggers
- Weekly emotional trading pattern summary
