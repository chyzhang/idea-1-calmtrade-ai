# CalmTrade AI

> A calmness layer for traders: breathe now, then get chart-context coaching.

## Problem

Users get anxious or impulsive while a trade is live.

## Solution

The app delivers breathing prompts while the user is trading.

## Target Users

Retail traders who make fast discretionary decisions on charts.

## Why People Want This

They want fewer impulsive trades and more consistent execution.

## Imagine If...

You’re about to revenge-trade; the app slows you down in 20 seconds.

## Replaces

White-knuckling emotions, generic meditation apps, and asking friends for quick chart takes.
